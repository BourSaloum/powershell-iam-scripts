
# PowerShell IAM Scripts

Scripts PowerShell simples pour automatiser des tâches IAM
sur Microsoft Entra ID.

## Contenu
- Joiner : création de comptes utilisateurs
- Leaver : désactivation de comptes
- Utilitaires : connexion Entra ID

Objectif : illustrer la compréhension du cycle de vie des identités (JML).
